# AstroVision Ultra PRO+ 🔭

iPhone 14 ve üstü cihazlar için optimize edilmiş profesyonel dijital dürbün ve teleskop PWA uygulaması.

## Özellikler
- **15X Dijital Zoom:** Donanım hızlandırmalı pürüzsüz yakınlaştırma.
- **Night Vision:** Yazılımsal düşük ışık güçlendirme.
- **HUD Telemetry:** Gerçek zamanlı açı ve koordinat takibi.
- **PWA Ready:** Ana ekrana eklendiğinde uygulama gibi çalışır.

## GitHub Pages Kurulumu
1. Bu repo içindeki `index.html` dosyasını kendi deponuzun ana dizinine yükleyin.
2. Repo ayarlarından **GitHub Pages**'i aktif edin.
3. Yayınlanan URL'yi iPhone Safari'de açıp "Ana Ekrana Ekle" deyin.
